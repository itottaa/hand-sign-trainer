HOW TO GET STARTED
================================================================================

OPTION 1: DEPLOY IMMEDIATELY (5 minutes)
─────────────────────────────────────────

1. Push code to GitHub:
   git init
   git add .
   git commit -m "Hand Sign Trainer"
   git remote add origin https://github.com/YOUR_USERNAME/hand-sign-trainer.git
   git branch -M main
   git push -u origin main

2. Deploy to Vercel:
   • Go to https://vercel.com/new
   • Select your GitHub repo
   • Click Deploy
   • Done! 🎉 App is live in 2-5 minutes

OPTION 2: TEST LOCALLY FIRST (10 minutes)
──────────────────────────────────────────

1. Install and run:
   pnpm install
   pnpm dev

2. Open http://localhost:3000

3. Test the app:
   • Collect some data
   • Train a model
   • Test recognition

4. Then deploy to Vercel (same as Option 1)

================================================================================
USER WORKFLOW
================================================================================

STEP 1: COLLECT DATA (Collect Data Tab)
────────────────────────────────────────
1. Enter a sign name (e.g., "Hello", "A", "Thank You")
2. Click "Start Collecting"
3. Show your hand to camera
4. System automatically records 30 samples
5. Repeat for 2-3 different signs

STEP 2: TRAIN MODEL (Train Model Tab)
──────────────────────────────────────
1. Click "🧠 Start Training"
2. Watch live metrics update
3. Wait for training (2-5 minutes)
4. Model is automatically saved

STEP 3: PRACTICE & RECOGNIZE (Practice Tab)
────────────────────────────────────────────
1. Click "▶ Start Practice"
2. Show your hand to camera
3. Recognition results appear ONLY when hand is visible ✨
4. See confidence score for each recognized sign

TROUBLESHOOTING
================================================================================

Issue: App loads but camera doesn't work
Solution: Ensure HTTPS (or localhost for dev)
          Check browser camera permissions
          Try different browser

Issue: Training is slow
Solution: Normal! Takes 2-5 minutes
          More samples = longer training
          CPU/GPU affects speed

Issue: Model accuracy is poor
Solution: Collect more samples (30+ per sign)
          Ensure good lighting
          Keep hand size consistent
          Train for more epochs

Issue: Deployment fails
Solution: Run 'pnpm install' locally first
          Check Node.js version compatibility
          See Vercel build logs

See DEPLOYMENT_CHECKLIST.md for more troubleshooting
